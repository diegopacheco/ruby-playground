# mruby Docker/Podman Image

Containerized mruby - lightweight implementation of the Ruby language.

## Build

```bash
./build.sh
```

## Run

```bash
./mruby.sh -e 'puts "Hello World!"'
./mruby.sh -e 'puts 1 + 2'
./mruby.sh script.rb
```

## Test

```bash
./test.sh
```
